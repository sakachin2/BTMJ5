종료된 게임, 중단된 게임 목록이 표시됩니다.
게임의 이력은 외부 기억 영역에, 없으면 앱의 데이터 폴더에 저장하고 있습니다. 앱의 데이터 폴더는 다시 배포하면 삭제됩니다. 외부 기억 영역은 Android-10까지는 SD 카드(/sdcard/eMahjong), Android-11 이후는 설치 직후의 기동으로 팝업한다 시스템 다이얼로그에서 사용자가 지정한 "Scoped Storage"가 됩니다.
늘어나면 적절하게 Delete하세요. 삭제의 경우 여러 줄을 선택할 수 있습니다(Multichoice). 선택 취소는 Deselect 버튼입니다. 다중 선택의 경우 다시 터치해도 해당 행의 선택이 취소됩니다.
B/S 버튼은 종료된 게임의 플레이어별 점수 합계를 표시합니다. Deselect 후 B/S 버튼을 누릅니다.
1행 선택하고 버튼을 누르면 그 게임과 같은 멤버의 게임의 집계를 실시합니다
중단 게임의 재개는 서버에서 행을 선택하고 Reload 버튼을 누른다. 규칙은 일시 중단시 사용했던 규칙을 사용합니다.
재개는 선택한 중단 게임과 같은 멤버(연결 대화상자에서 설정한 ID-Name)가 갖추어지지 않으면 할 수 없습니다.
서버에 중단된 게임의 기록을 저장할 수 없고, 클라이언트가 저장하고 있으면, 클라이언트의 히스토리로부터 재시작을 요구할 수도 있습니다. 클라이언트에서 행을 선택하고 Reload를 누르고 경고 대화 상자에 OK를 대답하십시오. 서버에 ResumeSuspend Confirmation 대화 상자가 표시됩니다.
Send 서버의 Send 버튼은 기록 행을 모든 연결 클라이언트로 보냅니다. 클라이언트의 Send 버튼은 서버로 보냅니다. 종국 게임의 확인 및 중단 게임의 서버로부터의 재개 요구를 위한 송신에도 사용됩니다. 수신자에게 경고 대화 상자가 표시되고 OK에서 저장됩니다.
