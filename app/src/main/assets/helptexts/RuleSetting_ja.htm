端数調整：100点端数処理
切り上げ、四捨五入、五捨六入はマイナスのときは以下のようになる
　四捨五入は -100 ～ -500 ===> 0, -600 ～ -900 ===> -1
　五捨六入は -100 ～ -400 ===> 0, -500 ～ -900 ===> -1
　切りあげは -100 ～ -900 ===> 0
　切り捨ては -100 ～ -900 ===> 0
合計をゼロにするために、端数調整による過不足は最終的にトップで調整する

ドラ： 和了画面に表示するのは表示牌です。 即とした場合、王牌を引いたときに表示されます。 そうでなければ打牌が成立したときです。 "槍槓"したときはカンドラはつきません。

トビ（ドボン）
ハコ下あり：ハコ下をも払って終局する。
順位確定後精算：ハコ下ありで徴収し、順位決定後、飛ばしたプレーヤーがハコ下分を返す。
ハコまでを得点とする：飛ばしたプレーヤーはハコまでしか得点を与えられない、 順位もその結果で決定する。
ダブロンありの時は上家どりに従って徴収しハコ下負担者が決まる。
流し満貫のときは放銃者はいないので、上家どりは東家あるいは東家に近い方から徴収する。

喰い変え：
ポン/チーしたときの打牌の制限です。
例えば、 の役無し聴牌しているとき、上家がを打牌したら一通を確定するためにチーして筋牌にあたる を捨てることができるか？
また でホンイツ聴牌しているとき、上家がを打牌して立直をかけてきた場合、 立直一発を避けるためにをチーして手牌中の現物を打牌できるか？

途中流局：
四人立直は四人目に捨て牌に一定時間 "和了" 宣言がないとダイアログがポップアップします。
四風連打は自動でダイアログがポップアップします。暗槓すると成立しませんが、メニューから途中流局を要求することはできます。
三家和は局の"終了確認"ダイアログで"確定"を押したときに途中流局ダイアログが開きます。 ダブロン無し(頭ハネ)でも流局とします。

ダブロン：
ロボット入りのゲームの時、赤（ロン用）の時間帯のなかでは人間優先でボットはロン宣言しません。 但し一人遊びの時はボットもロン宣言し、ダブロンルールに従います

ロボット：
一人遊びでない、ロボットを入れたゲームでは、ロボットの負けを他のプレーヤーが負担します。
均等割り：ロボットの負けの合計を均等割り、端数切捨ての発生の場合、合計ゼロの調整はトップ者で行います。
得点比例：ボット以外の最下位者の負担がゼロとなるように比例配分を行う。
　　例 合計点が +60000, +10000, -30000, -40000　でボットが4位:-40000 の場合
　　　6-(-3)==>11, 1-(-3)==>4, -3-(-3)==>0　の合計：15 に対する比例割り
　　　11/15*(-4)=-29300(100未満切り捨て), 4/15*(-4)=-10600, 0/15*(-4)=0
　　　合計ゼロ調整をした -29400, -10600, 0, +40000　を合計に加算すると
　　　精算値は +30600, -400, -30000, 0
