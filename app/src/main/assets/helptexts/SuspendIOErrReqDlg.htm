This dialog opens when detected ConnectionFailure.
Menu:Connection failure re-opens this dialog.
For WifiDirect connection,failure detection of both side may have some delay by a few minutes.
Reconnect button opens Blutooth Reconnection or Wi-Fi Direct reconnection dialog. Try reconnection.
If connection succeeded, server can Restart game.
If failed again, select Suspend/Gameover.
Depending timing of connection failure, countinued hand may be strange. You can re-call "Suspend/Gameover" from menu until end of the hand.
Suspend/Gameover opens SuspendGame by ConnectionFailure Claim dialog. Select Suspend or Gameover.
Suspend save the game with the current hand canceled.
Gameover also calculate score with the current hand canceled.
