종국이 되었을 때 "First Dealer"(起家)에, 그것이 로봇이라면 반시계 방향으로 다음 인간에게 이 다이얼로그가 표시됩니다.
누구나 다시 보기는 Final score 메뉴에서 할 수 있습니다.
Send 버튼을 누르면 다른 플레이어에게도 동일한 대화상자가 표시됩니다.
모든 사용자가 OK 응답하면 Fix 버튼으로 Gameover가 됩니다.
이름표에 포인트가 표시되며 메뉴의 Gameover에서 상단 화면으로 돌아갈 수 있습니다.
NG 응답이 있는 경우 경고 대화 상자가 표시됩니다.

S:Seat(좌석). 게임 시작 전 주사위로 결정한 좌석 순서. 동쪽이 "First Dealer".
Pl:Place(순위). Final의 순위.
TopPz:Top Prize. 1위 상. (30,000-InitialScore)*4. RuleSettings, InitialScore
PlcPz:PlacePrize. 4위가 1위에, 3위가 2위에 지불한다. RuleSettings, PlacePrize.
Debt:Tobi 상금. RuleSettings,Tobi,Tobi prize
Debt 계산 방법은 RuleSettings의 Help를 참조하십시오.
Final: 최종 점수.
ForRobot: 로봇의 부채를 인간이 부담한다. ForRobot 계산 방법은 RuleSettings의 Help를 참조하십시오.
Point : (Final - 30,000) / 1000.
