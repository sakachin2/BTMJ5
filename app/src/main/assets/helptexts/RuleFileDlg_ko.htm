RuleSettings 파일을 조작합니다.
각 대국 멤버 그룹에 대해 규칙을 구분하기 위해 규칙 파일의 이름을 지정하고 저장하고 로드합니다.
이름의 뒷부분이 "sg"인 것은 중단된 게임에서 사용된 규칙 파일입니다. 목록의 뒷부분에 함께 표시되며 중단 게임을 다시 시작하는 데 사용됩니다.
규칙 파일은 외부 저장소에 저장됩니다. 외부 저장소를 사용할 수 없는 경우 앱의 데이터 폴더에 저장합니다. 앱의 데이터 폴더는 다시 배포할 때 삭제됩니다. 외부 기억 영역은 Android-10까지는 SD 카드(/sdcard/eMahjong), Android-11 이후는 설치 직후의 기동으로 팝업한다 시스템 다이얼로그에서 사용자가 지정한 "Scoped Storage"가 됩니다.

Multi: 여러 줄 선택 모드로 설정합니다. 일괄 삭제할 때 사용합니다.
다중 행 선택 모드에서 다시 터치하면 선택이 취소됩니다.
SelectAll: 모든 항목을 삭제할 때 사용
Reset: 모든 선택을 취소합니다.
EXtension: Load일 때 표시됩니다.
Name: Save일 때 표시됩니다. RuleSettings 대화 상자에서 설정한 ID가 기본적으로 설정됩니다. 필요한 경우 파일 이름을 변경하십시오.
Delete: 선택한 파일을 삭제합니다.
Open: 파일 내용을 표시하지만 개발자용입니다.
