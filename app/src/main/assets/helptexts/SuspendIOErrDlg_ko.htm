ConnectionFailure Notification에서 Suspend/Gameover 버튼을 누르면 이 대화상자가 표시됩니다.
실수로 닫았거나 처음부터 다시 시작하려면 메뉴에서 Connection failure를 누릅니다.
Gameover score가 표시됩니다. Suspend 또는 Gameover를 선택하세요. 두 경우 모두 현재 라운드는 무시됩니다.
PlayerConnectionFailed에는 설정 벌금이 부과됩니다. 절단의 원인은 어느 쪽에 있는지, 어느 쪽에 벌금을 부과할까는 토론해 주세요.
Pao(包)를 도망치려고 했다고 판단되는 경우는, Penalty as Pao에 체크를 넣어 주세요. RuleSetting,SuspendOption 의 Penalty for ConnectionErr 와 큰 쪽이 부과됩니다.
서버에서 Send를 누르면 연결이 남아 있는 클라이언트에 동일한 내용이 표시됩니다. 클라이언트는 OK/NG를 반환합니다. 연결이 끊어지지 않은 모든 클라이언트의 응답이 있으면 Fix 버튼으로 Suspend 처리를 완료합니다.
재연결할 수 없는 클라이언트는 각 클라이언트에서 Suspend 처리하고 재개에 대비하여 기록을 저장해 두십시오. Fix 버튼을 사용합니다.
Fix 후 메뉴의 Gameover에서 톱 화면으로 돌아가세요. Suspend한 게임은 Records 버튼에서 나중에 다시 시작할 수 있습니다.
