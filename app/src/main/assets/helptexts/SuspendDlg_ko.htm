"End-of-Hand", "ExhaustiveDraw Confirmation", "AbortiveDraw Confirmation" 대화상자에서 Suspend/Gameover 옵션을 켜면, "Fix" 버튼을 누를 때 표시됩니다. 단 "Final Hand"의 경우는 "On FinalHand"대화 상자가 표시됩니다.
실수로 닫혀 있거나 작업을 다시 시작하려면 메뉴에서 Suspend Dialog를 누르십시오.
정산할 경우의 점수 상황을 표시하고 있습니다. Suspend, Gameover 또는 Continue를 선택하세요.
Gameover의 경우 Suspender에는 설정 벌금이 부과됩니다. 벌금은 Related에서 확인하세요. (설정은 RuleSettings,Operaions,SuspendOption).
Continue 경우 Suspend/Gameover가 취소되고 게임이 계속됩니다.
Send를 누르면 모든 클라이언트에 동일한 내용이 표시됩니다. 클라이언트는 OK/NG를 반환합니다. 모든 클라이언트의 OK 응답이 있으면 Fix 버튼으로 게임 중단이 완료됩니다.
NG 응답이 있으면 경고 대화 상자가 표시되므로 이를 따르십시오.

Suspend 또는 Gameover를 선택한 경우 메뉴의 Gamover에서 상단 화면으로 돌아가세요.
